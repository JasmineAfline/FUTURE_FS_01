# Personal Portfolio Website

This repository contains a simple personal portfolio website built with plain HTML, CSS, and JavaScript.

## ✅ What’s Included

- **SEO-friendly structure** with proper metadata and Open Graph tags.
- **Interactive sections** (About, Projects, Skills, Contact).
- **Mobile-friendly navigation** with a responsive hamburger menu.
- **Contact form UI** with placeholder logic for later backend integration.
- **Project listing** rendered via JavaScript for easy updates.

## 🚀 How to Run Locally

### Option 1: Open directly
1. Open `index.html` in your browser.

### Option 2: Run a simple local server (recommended)
From the repo root:

```bash
# If you have Python 3 installed
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
